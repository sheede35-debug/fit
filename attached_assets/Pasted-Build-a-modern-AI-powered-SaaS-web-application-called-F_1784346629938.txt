Build a modern AI-powered SaaS web application called "FlowIQ".

Project Purpose:
FlowIQ is an intelligent workflow management platform that connects all company departments in one centralized system. It tracks every request from creation to completion while using AI to analyze bottlenecks, predict delays, optimize workflows, and provide executives with actionable insights.

Design Requirements:
- Premium enterprise UI similar to Linear, Notion, Stripe Dashboard, and Vercel.
- Clean, minimal, modern interface.
- Rounded cards.
- Glassmorphism effects.
- Soft shadows.
- Professional typography.
- Responsive for desktop, tablet, and mobile.
- Smooth animations and page transitions.
- Dark Mode and Light Mode.
- High-quality charts and dashboards.

Authentication:
- Company Registration.
- Login.
- Forgot Password.
- Multi-tenant architecture (each company has isolated data).
- Roles:
  - Super Admin

  - Manager

  - Employee

AI Features:

1. AI Delay Prediction
Predict:
- Estimated completion date
- Probability of delay
- Estimated remaining days

Example:
Delay Probability: 83%
Expected Completion: 4 Days

2. AI Bottleneck Detection
Automatically identify:
- Departments causing delays
- Employees with overload
- Most delayed workflow
- Peak workload times

3. AI Root Cause Analysis
Analyze why requests are delayed.

Examples:
- Missing documents
- Waiting for approval
- High employee workload
- Incomplete information
- Cross-department dependency

Display confidence score.

4. AI Workflow Optimizer
Analyze workflow efficiency.

Suggest:
- Remove unnecessary approval steps
- Reorder departments
- Merge duplicate approvals
- Reduce completion time

Display estimated improvement percentage.

5. AI Smart Prioritization
Automatically prioritize requests based on:
- Due date
- Client importance
- Business impact
- Delay risk
- Request type

6. AI Auto Classification
Read request description using AI.

Automatically determine:
- Request category
- Priority
- Required departments
- Expected completion time

7. AI Document Validation
Analyze uploaded documents.

Detect:
- Missing signatures
- Missing attachments
- Incorrect formats
- Incomplete forms

Provide suggestions before submission.

8. AI Executive Assistant
Create a built-in AI chatbot.

Managers can ask:
"Why is Request #102 delayed?"

"What department is causing the most delays?"

"Show this week's performance."

The AI answers using system data.

9. AI Weekly Executive Summary
Automatically generate executive reports.

Include:
- Best performing department
- Slowest department
- Number of completed requests
- Number of delayed requests
- Main delay causes
- Performance trends
- AI recommendations

10. AI What-if Simulation
Allow managers to simulate improvements.

Example:
"What happens if Finance gets two additional employees?"

AI estimates:
- Reduced completion time
- Reduced bottlenecks
- Increased SLA compliance
